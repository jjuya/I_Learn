# 0_Initialize an npm Project



### Project Setup

1. 빈 프로젝트 생성

   ```bash
   mkdir react-bolilerplate
   ```

2. 프로젝트 초기화

   `npm init` or `yarn init`



### Git Setup

1. `git init`
2. `git status`
3. `git add [file name]`
4. `git commit -m [message]`
5. `git push origin master`
   - origin 설정 : `git remote origin [repository url]`
