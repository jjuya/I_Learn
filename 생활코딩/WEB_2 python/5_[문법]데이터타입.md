### List

- 데이터가 많아져서 복잡해질 때 수납할 수 있는 공간
- https://docs.python.org/3/tutorial/datastructures.html
- https://docs.python.org/3/library/stdtypes.html#sequence-types-list-tuple-range

```python
s = [1, 'four', 9, 16, 25]
print(s)
print(s[1])
print(len(s))
s[1] = 4
print(s)
del s[2]
print(s)
s.append('egoing')
print(s)
```
