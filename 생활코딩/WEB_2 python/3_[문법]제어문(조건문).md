## 제어문(조건문)

### 제어문

- 프로그래밍은 컴퓨터가 해야할 일을 순서대로 실행되도록 하는 것
- 그렇게 만들어진 순서들은 프로그램



#### 1. 제어문

- 제어문은 순서대로 실행하는 중에 흐름을 바꾼다.

- 조건문 : 조건에 따라 다른 일을 실행
- 반복문 : 조건이 만족될 때 일을 반복



#### 2. index.py 수정

- index.py를 실행하였을 때 id 값이 없어 에러가 발생
- 임시 방편으로 수정



- `index_no_id.py`를 생성
  - `index.py`를 복사
  - pageID를 'Welcome'으로 지정
  - 리눅스나 맥의 경우 실행 권한 추가 (`sudo chmod a+x index_no_id.py` )

