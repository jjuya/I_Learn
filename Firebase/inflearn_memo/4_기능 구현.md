## 기능 구현

### 인증 기능 구현

#### 1. 구글 로그인 창 띄우기

```js
var auth = firebase.auth();
var authProvider = new firebase.auth.GoogleAuthProvider();

auth.signInWithPopup(authProvider);
```

- `firebase serve`를 통해 확인



#### 2. 성공/실패 나누기

```js
var auth = firebase.auth();
var authProvider = new firebase.auth.GoogleAuthProvider();

auth.onAuthStateChanged(function(user) {
  if(user) {
    // 인증 성공
    console.log("sucess!");

    // 메모 리스트 출력
  } else {
    // 인증 실패
    auth.signInWithPopup(authProvider);
    console.log("false!");
  }
});
```
